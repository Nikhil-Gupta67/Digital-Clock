# Modern Clock

A minimal, modern-styled digital clock built with HTML, CSS and JavaScript.

## Features

- Clean, responsive design
- Real-time digital clock with smooth updates
- Single-file front-end (no build step)

## Files

- [index.html](index.html) — markup and demo page
- [index.css](index.css) — styles and layout
- [index.js](index.js) — clock logic

## Usage

1. Open [index.html](index.html) in your browser to view the clock.
2. (Optional) Serve locally for consistent behavior:

```bash
python -m http.server 8000
# then open http://localhost:8000
```

## Customization

- Edit the CSS in [index.css](index.css) to change colors, fonts, or layout.
- Edit the JS in [index.js](index.js) to modify time format or add features.
